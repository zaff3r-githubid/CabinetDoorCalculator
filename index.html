Cabinet door guide & reference

Anatomy of a 5-piece door

Flat- and raised-panel doors are built from 5 parts: 2 stiles, 2 rails, and 1 panel. Stiles run the full height; rails fit between them. Both carry a centered groove that captures the panel, letting it swell and shrink with humidity without cracking the frame.

Door styles & how they fit the opening

The style sets whether you add overlay or subtract clearance from the opening — so pick it first.

  • Inset — sits flush inside the frame with a ~1/16" gap all around. Subtract clearance.
  • Lipped — a rabbet around the edge lets it partially recess; overlays ~3/8".
  • Partial overlay — sits proud, overlaps the frame ~1/2" per side (the app's default).
  • Full overlay — covers the frame almost entirely, so no face frame shows.

Measuring the opening

Measure the inside clear opening of the face frame, not the overall cabinet. Check width at top & bottom and height on both sides; use the smaller reading if the frame is out of square. Then add overlay (or subtract clearance) to each edge.

The formulas

  • Single overlay door: Door W = Opening W + (2 × Overlay), Door H = Opening H + (2 × Overlay)
  • Double (pair) overlay: Door W = [(Opening W + 2×Overlay) − Center Gap] ÷ 2
  • Inset single: Door W = Opening W − (2 × Clearance)
  • Rail length: Door W − (2 × Stile W) + (2 × Groove Depth)
  • Panel width: Door W − (2 × Stile W) + (2 × Groove Depth) − (2 × Expansion Gap)  (height uses the same method)

Worked example

18" wide door · 2¼" stiles · 3/8" groove · 1/16" expansion

Rail width = 18 − (2.25 × 2) + (0.375 × 2) = 14.25"
Panel width = 14.25 − (0.0625 × 2) = 14.125"

Joinery — the four common corner profiles

All 5-piece doors need a groove for the panel; the difference is the profile milled on the inside edge where rail meets stile.

  • 90° / Shaker — a plain square edge with a groove. No profile bits needed; just a table saw or straight router bit, plus a tenon on the rails.
  • Bevel, Ogee, Radius — decorative profiles cut with matched cope-and-stick router bits: one bit mills the groove & profile on rails and stiles, the other cuts the matching tenon on the rail ends.

Frameless (Euro / 32mm) cabinets

Frameless cabinets have no face frame — the door mounts straight onto the box (carcass) side panels using concealed 32mm-system hinges. Because there's no frame to overlay or sit inside of, the measurement logic is closer to Inset: you measure the outer cabinet box dimensions, and the door is sized just slightly smaller than the box, leaving a small reveal gap all around and between paired doors.

  • Cabinet Box Width / Height — the outer dimensions of the box (side panel to side panel, top to bottom), not an inside frame opening.
  • Reveal Gap — the gap between the door edge and the box edge, between a pair of doors, and between stacked drawer fronts. Typically 1/16"–1/8" (1.5–3 mm).
  • Door W = Box W − (2 × Reveal), same formula the app uses for Inset — just applied to the box instead of a frame opening.
  • Hinges are concealed cup hinges bored into the door's back (usually 35mm cup), screwed to a mounting plate on the box side — different hardware than framed-cabinet butt/partial-overlay hinges.

Select Frameless (Euro / 32mm) under Cabinet Style to switch the width/height/gap fields to this mode. 5-piece Shaker doors work fine on frameless boxes — very common combination in modern kitchens — only the outer sizing math changes, not the door construction.

Drawer fronts

Drawer fronts are calculated as flat slab panels (the standard approach — most drawer fronts, including in Shaker kitchens, are too short for a rail/stile/groove assembly). Set Number of Drawer Fronts and the Drawer Bank Opening Height to size a stack: the app applies the same overlay/inset/frameless formula to get the outer stack size, then splits it evenly across the fronts with a reveal gap between each one. Drawer front width always spans the full opening, matching the door(s) below it. Slab area is added to the sheet goods estimate; hardware adds one pair of slides per drawer.

Metric mode converts every value using 1 in = 25.4 mm. Imperial shows to the nearest 1/16"; metric to the nearest 0.5 mm.
